StellarBeats Privacy Policy

Effective Date: February 2026

Thank you for using StellarBeats! This Privacy Policy explains how we collect, use, store, and protect data when you use the StellarBeats Discord bot. By using StellarBeats, you agree to the practices described below.

Data Collection

StellarBeats is a music-focused Discord bot designed to provide smooth and high-quality audio playback. To function correctly, StellarBeats collects and stores limited, non-personal data, including:

  • Server IDs – to manage server-specific music settings
  • User IDs – for command handling, premium access (e.g., No Prefix), and abuse prevention
  • Voice Channel IDs – to connect, play, and manage music sessions
  • Bot Configuration Data – such as autoplay, filters, volume, and feature toggles
  • Command Usage Data – used to improve performance and detect misuse

StellarBeats does not permanently store message content or collect personal or sensitive information.

Use of Data

Collected data is used only for the following purposes:

  • Providing music playback and related features
  • Managing premium features and subscriptions
  • Preventing abuse, spam, or exploit attempts
  • Improving bot stability, performance, and user experience
  • Debugging issues and providing support

We never sell, rent, or share user or server data with third parties.

Message & Command Processing

  • Commands are processed in real time to execute requested actions
  • Message content is not stored permanently
  • Only minimal metadata (such as command type or user ID) may be logged for error handling or abuse prevention

Data Security

  • Reasonable security measures are used to protect stored data
  • Data is kept only as long as required for functionality
  • Unnecessary data is periodically removed

Access to Data

  • Server owners and administrators control how StellarBeats is configured in their server
  • Bot developers may access limited data only for maintenance, debugging, or feature improvement

Policy Updates

This Privacy Policy may be updated at any time. Continued use of StellarBeats after updates means you accept the revised policy.

Support & Contact

If you have questions, concerns, or data-related requests, please contact us via the StellarBeats Support Server
